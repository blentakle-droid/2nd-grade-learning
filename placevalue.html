Place value

Learn place value up to 100

Place value is the value of where a digit is in a number.


Example, 477

  1. The 4 is in the hundereds place, so it is worth 400
  2. One of the 7 is in the tens place, so it is worth 70
  3. The other 7 is in the ones place, so it is worth just 7
  4. So 477 is made up of 400 + 70 + 7

C, because it shows that the 8 is worth 800, the 1 is worth 10, and the 9 is worth 9

4) 0345 = ?

A, because even if order isn't perfect, it still shows that the 0 is worth 0, the 3 is worth 300, the 4 is worth 40 and the 5 is worth 5
